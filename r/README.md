# R Analysis Scripts

This directory contains the main R scripts used for thesis-related analysis.

## Structure

- `thesis_results/`: main R scripts used for result generation in the final thesis
- `src/main/R/`: auxiliary or scenario-related R scripts from the project structure

## Note

The included scripts represent the main analysis workflow used in the thesis.
Some intermediate exploratory scripts from earlier development stages are not fully preserved or were kept in older repositories.
